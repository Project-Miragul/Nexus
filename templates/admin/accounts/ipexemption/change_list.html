How IP Exemptions Work

The server enforces the box limit above for every incoming game client connection. If the connecting IP has a row in this table, exemption_amount overrides the global rule for that IP only — everyone else still hits the standard limit.

Common case: Multiple people in the same household share one external IP (router NAT). They're all separate real players but the server can't tell them apart. Add their external IP here with exemption_amount set to the number of people in the house.
